Is CentOS…dead?

CentOS is a Linux distribution that provides a free and open-source community-supported computing platform, functionally compatible with its upstream source, Red Hat Enterprise Linux (RHEL). In December 2020, Red Hat unilaterally terminated CentOS development. In response, CentOS founder, Gregory Kurtzer, created the Rocky Linux project as a successor to the original mission of CentOS.

Rocky Linux

Rocky Linux is a Linux distribution that is intended to be a downstream, complete binary-compatible release using the Red Hat Enterprise Linux (RHEL) operating system source code.

Software foundation

Completely outside of the Red Hat camp—just like CentOS used to be—Rocky Linux is governed and administered by the Rocky Enterprise Software Foundation. Seven months after announcing the project, the first GA version of Rocky Linux was released.

Installation

Selecting “Install Rocky Linux 8” from the installation menu displays a series of familiar screens. Choose your keyboard and regional settings, set a root password and choose your partitioning scheme and the part of the installation that copies the files starts up.

Restart and you will see the Rocky Linux login screen. Once logged in, a familiar and minimalist GNOME 3.32.2 desktop welcomes you.

Easy, not?

Other possibilities?

Rocky Linux isn’t the only way to replace CentOS though. Alma Linux, developed by Cloud Linux, is an open-source operating system that is 1: 1 binary compatible with RHEL and is supported by the community.

Another distro is Oracle Linux; is engineered to provide reliability, exceptional performance, and security for open cloud infrastructure. As pointed out early, it’s completely free to download, use and redistribute. It is freely distributed by Oracle and made available under a partial GNU GPL license.

Conclusion

Big names are also eager to be associated with Rocky Linux. Google is a “main sponsor” of the RESF and has built a custom Rocky Linux for the Google Compute Engine. Microsoft has signed up as a RESF partner and is committed to ensuring that Rocky Linux is available on its Azure platform.

A project is not just about software. The Rocky Linux project has spent a long time deciding and formalizing its internal structure, governance and ethics.

To you the choice.
