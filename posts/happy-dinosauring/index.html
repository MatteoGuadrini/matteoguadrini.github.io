Fun terminal tools

There are very useful command line tools, which are used for daily administration, whatever the job you have to do, an example are find, grep, cat, less, etc. There are also more complex tools that require the study and consultation of manuals such as docker, vagrant, etc. and are for advanced users such as systems engineers and developers.

Other tools are used … well, practically nothing, except to elicit a laugh.

dinosay, is one of these tool.

Print dinosaur

Basically, dinosay is useless except to print dinosaurs on the terminal with a message.

Follow this for installation:

$ pip install dinosay       # for python enviroment

$ dnf copr enable matteoguadrini/dinosay
$ dnf install python-dinosay -y    # for Red Hat, CentOS, Mageia and fedora
$ wget https://github.com/MatteoGuadrini/dinosay/releases/download/<version>/dinosay-<version>-1.noarch.rpm -O dinosay.rpm
$ yast -i dinosay.rpm       # for SLES and OpenSUSE

$ git clone https://github.com/MatteoGuadrini/dinosay.git && cd dinosay
$ python setup.py install   # for other

Now, print a test trex dinosaur:

$ dinosay -d trex

Without message, dinosay print a default message “Rooooaaaaarrrr”!

Change dinosaur

To change dinosaur, just write its name or alias as the argument of the -d parameter:

$ dinosay -d dim "I'm a dimetrodon!"
$ dinosay -d maia "Now, I'm maiasaur"

There is a random mode that chooses the dinosaur for you. Just specify the -r parameter:

$ dinosay -r "Woooooooo...I'm...???"

Behavior, eyes and tongue

It is possible to change some elements of each printed dinosaur. It is possible to change the balloon containing the message via the parameter -b which controls the behavior of the dinosaur. The list of behaviors can be obtained via the dinolist command:

$ dinosay -d trex -b happy "I'm happy!"

This command modifies both the $eye variable and the comic.

To change only the $ eye element instead, just look at the types of eyes available in the dinolist command to specify them in the -e parameter:

$ dinosay -d trice -e glass "I have a glass!"

Language can also be added for some dinosaurs. Test this dinosaur by specifying the -t command as well:

$ dinosay -d maya -t "Bleeeeehhhhhh!"

If you want to give your dinosaur a “thinking” look, just specify the -i (idea) parameter to make a comic appear as if it were thinking:

$ dinosay -d plei -i "I thinking about...dinosay!"

Color everything

There is a special parameter that allows you to take advantage of the console colors to add that extra touch to your ASCII dinosaur. Test your dinosay commands with -c:

$ dinosay -r -c cyan
$ dinosay -r -c darkcyan
$ dinosay -r -c blue
$ dinosay -r -c red
$ dinosay -r -c yellow
$ dinosay -r -c green
$ dinosay -r -c purple

Wrap text message

If your message is too long, then you can use the -w parameter to delineate the maximum number of characters per column:

$ dinosay -r -w 20 "This is a long long long long message wrapped in twenty character columns!"

Load a file

If you would like to build your dinosaur or animal (or whatever you want) in ASCII, you can use the -f parameter to specify your ASCII file. You can also use the variables usable by dinosay in your file to replace eyes, language and message (see the docs:

$ dinosay -f /home/test_dino/mydino.ascii "My dinosaur!"

dinosay like a python module

dinosay is written in python and can also be used in python scripts as an import module.

from dinosay import dinoprint, Dino
body = r"""
           __n__n__
    .------`-\$eyes/-'
   /  ##  ## (oo)
  / \## __   ./
     |//YY \|/
     |||   |||
"""
my_dino = Dino(body, "Hi, I'm alternative of cowsay", behavior='happy')
dinoprint(my_dino.message, my_dino.body, my_dino.behavior)

# Now in color
my_dino.color = 'red'
my_dino.apply_color()
dinoprint(my_dino.message, my_dino.body, my_dino.behavior)

Conclusion

dinosay is a useless tool, but it is a lot of fun for creating intro banner shells or introducing sympathy into your bash and python scripts, using it as a command line or as a python module. Happy dinosauring!
