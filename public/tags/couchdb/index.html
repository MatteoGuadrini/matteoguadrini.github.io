What powershell? Powershell is an object-based shell born in Microsoft in 2006, made open source 10 years later. It is characterized by commands, called cmdlets, which respect a certain pattern. An example cmdlet is as follows: Get-ChildItem -Path value The pattern in question is Verb-Noun -Option argument, where Verb is a list of verbs.
